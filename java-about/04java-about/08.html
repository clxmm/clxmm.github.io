# idea 的一些问题

# 1，更改maven 导入的的jdk设置，

要设置为自己本机安装的jdk，不是本机安装的有可能下载不成

# 无法下载源代码 未找到此对象的源: 的问题

打开idea的终端 ，在本项目目录下，执行以下命令

mvn dependency:resolve -Dclassifier=sources
